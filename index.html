Floating Monster: Uncompromising Multichain Security

The Problem with Multichain

Most multichain projects compromise on security. This is because truly decentralized cross-chain communication is impossible with independent ledgers.

The Floating Monster Solution

Floating Monster takes a different approach. Our token exists simultaneously on all connected blockchains, creating a locked state that can only be altered by unanimous consensus across all ledgers.

Unparalleled Security

While this makes transactions in the locked state expensive, it also provides the combined security of all participating blockchains.

Flexibility for Everyday Use

For everyday use, tokens can be unlocked to a single ledger, functioning like any other token on that chain. Cross-chain transactions are also possible, with nodes incentivized through fees and required to lock assets on the destination ledger for added security.
